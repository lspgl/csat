Walker.scan(self, skeleton, maxwidth=10, nbands=inf)
Scan the skeleton starting from the top left corner Parameters:
skeleton: 2D array
skeletonized image
maxwidth: int, optional
maximal width of a scanning line within which a band is considered to be connected. Default is 10px
nbands: int, optional
number of bands to be scanned before the scan is interrupted or the end of the image is reached. Default is infinite
Returns:
coords: List of tuples of coordinates

endpoints: List of tuples of coordinates

Coordinates of band terminations which are not on an image boundary
Walker.skeletonize(self, img)
Morphological skeletonization of the binary image The image is eroded, dilated and subtracted before being compared with a logical or until one step before the 0-image is obtained. The resulting image is the skeleton. Parameters:
img: 2D array
Binary image to be skeletonized
Returns:
skeleton: 2D array
skeletonized image
